Вопрос №1: Подключение к SQL-базам в Python
Подключение представлено обьектами подключения, которые являются интерфейсом 
для различных операций и скрывают сложности связанные с особенностями отдельных БД. 
Также они могут использоваться для создания курсоров. Пример создания такого обьекта для 
SQLite:
import sqlite3
connection = sqlite3.connect('database loc')

можно сразу получить курсор из соединения, чтобы сразу начать выполнять выражения: 
c = sqlite3.connect('database loc').cursor()
c.exectute('SELECT * FROM INFORMATION_SCHEMA.tables')
