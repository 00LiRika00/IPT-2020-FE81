Одна з основних причин того, що Python - продуктивніша мова, - динамічна типізація. Це означає, що нам не потрібно нічого оголошувати - ми просто задаємо ім'я змінної і присвоюємо значення. Python самостійно визначає її тип відповідно до присвоєних значень.

А ось Java - статично типізована мова. Всі типи змінних тут повинні бути оголошені. Якщо допустити помилку, то програма працювати не буде або буде, але з проблемами.
Найпростішої програма. Java вимагає багато букв для виведення одного рядка. Це перший приклад, в якому наочно демонструється лаконічність Python.
JAVA
public class Main {
  public static void main(String[] args) {
     System.out.println("hello world");
   }
}

PYTHON
	
print("hello world");

Щоб ми не робили в Java, нам потрібно почати з створення класу, який буде містити необхідні методи.
