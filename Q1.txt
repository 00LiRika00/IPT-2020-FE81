# Аргументы Фукций на языке Python. Понятие *args и **kwargs.
# Вопрос №2: 
'''
Функции в Python:
def func (arg1, arg2,... argN):
    ....
    ....
    ....

- Аргументы функций на языке Python принимаются в упорядоченом слева-на-право
перечне;

- Аргументы по умолчанию (по типу arg1=2, arg2=5) при вызове функции меняются 
  на те аргументы с которыми вызывалась эта функция. Если при вызове функции 
  значения не присваивались - значения остаются по умолчанию;

- Использование *args и **kwargs в качестве аргументов функции применимо в том случае, 
  когда заранее не известно количество аргументов при вызове функции;

* Подробнее:
  *args — сокращение от «arguments» ( (именованные) аргументы), 
  **kwargs — сокращение от «keyword arguments» (именованные аргументы).

  *args нужен, когда мы хотим передать неизвестное количество неименованных аргументов - кортежи;
  **kwargs нужен, когда мы хотим передать переменное количество именованных аргументов - словари;

  def func (*args, **kwargs):
    ....
    ....
    ....
*    
'''
