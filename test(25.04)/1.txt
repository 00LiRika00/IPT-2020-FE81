В Python доступны следующие интерфейсы баз данных:
MySQL, Oracle, Sybase, Informix, InterBase, PostgreSQL, SQLite, ODBC.
Все взаимодействие основывается на понятии объектов:
соединений, курсоров, результатов запросов SQL. С помощью 
методов етих объектов выполняется все инструкции SQL для работы с базой.
