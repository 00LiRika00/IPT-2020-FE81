1. Модуль crossenv представляет собой инструмент для кросс-компиляции в языке Python. 
Он создает специальную виртуальную среду, такую, чтобы пакет-менеджер pip или setup.py 
смогли кросс-компилировать пакеты програмного кода.

Cross-python сообщает всю системную информацию. Он действует как Host-python. 
Инструменты distutils и setuptools будут кросс-компилироваться при сборке пакетов, и проблем  с упаковочным 
оборудованием не будет.

2. Nuitka - это (транспайлер) компилятор исходного кода, который компилирует код Python в исполняемые файлы C/C ++ или исходный код C/C ++. 
Он принимает код Python различных версий Python (в настоящее время 2.6, 2.7, 3.2, 3.3, 3.4, 3.5 и 3.6), и позволяет создавать 
автономные программы, даже если Python не установлен на целевом компьютере. Nuitka выпущена под лицензией Apache.

Он переводит Python в программу уровня C, которая затем использует libpython и несколько собственных файлов C для 
выполнения програмного кода таким же образом, как это делает CPython. Вся оптимизация направлена на то, 
чтобы избежать накладных расходов там, где это не нужно. Ни один из них не нацелен на удаление совместимости, 
хотя в будущем разработчики обещают улучшения, когда эмулируется не каждая ошибка стандартного Python, а
например более полные сообщения об ошибках данных.
