File Transfer Protocol — протокол передачи файлов по сети. 
Широко используется для распространения ПО и доступа к удалённым хостам. 

Python использует библиотеку ftplib для работы с протоколом. 
Средствами данной библиотеки, можно подключается с удалённым сервером для обмена данными. 
Для этого необходимо знать порт на котором находится сервер (по умолчанию FTP работает на 21-м порту). 
Так же требуется введение учетных данных для входа в систему, то есть вашего имени пользователя и пароля, 
чтобы получить доступ к файлам на сервере. После удачного входа - можно воспроизводить манипуляции на стороне клиента: 
перемещение по каталогам, отправка данных и т.д.
