Генераторы -- это функции которые возвращают объект, который можно итерировать. Их можно приостанавливать и возобновлять во время их выполнения. Таким образом, они намного эффективнее используют память при работе с большими наборами данных. Получить их значение можно лишь один раз. Это связано с тем, что они не хранят значения в памяти, а генерируют их. 

Yield -- это ключевое слово, которое используется как return — отличие в том, что функция вернёт генератор.
